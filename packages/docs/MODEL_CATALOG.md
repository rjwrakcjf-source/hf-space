# Model Catalog

Complete documentation for the HF Space model catalog.

## Categories

### Text Generation
Models for generating human-like text.
- GPT-2 (117M parameters)
- DistilGPT-2 (82M parameters)
- GPT-Neo 125M

### Code Generation
Specialized models for code.
- Starcoder2-3b
- CodeLlama-7b

### Embeddings
Vector embedding models.
- all-MiniLM-L6-v2
- all-mpnet-base-v2

### Vision
Image understanding models.
- CLIP ViT-B/32
- ResNet-50

## Adding Models

To add a model to the catalog, edit `models/catalog.json`.

## Model Configuration

See `models/config/` for routing, fallback, and performance configurations.
