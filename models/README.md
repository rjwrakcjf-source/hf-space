# Models Catalog

ML model catalog for HF Space with adapters and configuration.

## Adding a Model

Edit `catalog.json` and add a new entry.

## Adapters

- `huggingface-adapter.js` - Hugging Face Hub
- `onnx-adapter.js` - ONNX Runtime
- `tensorflow-lite-adapter.js` - TF Lite
- `ggml-adapter.js` - GGML/llama.cpp
